Full Colour Alerts

Add full colour using .bg-*. You may add .text-white when necessary. Use .alert-icon-full instead of .alert-icon to compliment the body colour better.

{{< alert_icons_full.inline >}} {{- with $.Page.Params.theme_colors -}} {{- range . -}}

Icon Alerts

Add bg-body text-body border-0 to get a coloured icon with white body and no border. You may also want to use .link-* instead of .alert-link for links.

{{< alert_icons_custom3.inline >}} {{- with $.Page.Params.theme_colors -}} {{- range . -}} {{ $linkColor := (cond (and (ne .name "light") (ne .name "dark")) .name "body" ) }}
{{ .title }}! A simple alert — check it out. Link Text
{{- end -}} {{- end -}} {{< /alert_icons_custom3.inline >}}

Border Icon Alerts

Add .bg-body .text-body and .border-* to get a white body with coloured border. You may add .text-bg-* to the .icon element so it matches the border colour.

{{< alert_icons_custom4.inline >}} {{- with $.Page.Params.theme_colors -}} {{- range . -}} {{ $linkColor := (cond (and (ne .name "light") (ne .name "dark")) .name "body" ) }}
{{ .title }}! A simple alert — check it out. Link Text
{{- end -}} {{- end -}} {{< /alert_icons_custom4.inline >}}
