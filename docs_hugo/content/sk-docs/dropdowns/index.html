By default, Custom Dropdowns are disabled. Enable Custom Dropdowns by setting $enable-dropdown-colors to true in Scss. When enabled, change colours using .dropdown-menu-{theme-color}.

Set or remove theme maps by using $dropdown-colors if only certain colours are needed. Remove light and dark to avoid duplicate styles as Bootstrap creates these by default. The same goes for secondary as this is the same as light unless you are using a custom colour.
