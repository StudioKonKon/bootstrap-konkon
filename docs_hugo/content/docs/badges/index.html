Hover and focus styles for a.badge and button.badge were removed from Bootstrap starting v5.0.0-alpha1.

Instead, we are using .badge-link .btn-* and .badge-link .btn-outline-* as an alternative by taking advantage of the css variables. Do this without using the .btn class. Button badges will need to use .badge-btn or .badge-btn-outline to work properly.
