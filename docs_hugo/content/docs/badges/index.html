Themed Badges

Hover and focus styles for a.badge and button.badge were removed from Bootstrap starting v5.0.0-alpha1. Instead, here above, we are using .btn .btn-* as an alternative. Using .btn-outline-* with badges, however, is not supported.

Below, we have included our own custom styles for badges. You can use the badge-* class to properly style borders, anchors and buttons without the need of using too many utility classes or depending on .btn which would cause issues with some layouts.

Custom badges are not required and can be disabled by setting $enable-custom-badges to false in Sass, if you don't need these or are fine with Bootstraps own default styles.
